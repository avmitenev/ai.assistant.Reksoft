
# Экспертиза по Java от «Рексофт» в курсе онлайн\-университета Skypro



**11 апреля 2022 года**. Онлайн\-университет Skypro, входящий в группу Skyeng, привлек экспертизу «Рексофт» (Reksoft), одного из ведущих российских разработчиков цифровых решений, в свой новый [курс по обучению Java\-разработчиков](https://sky.pro/courses/programming/java-developer).


«Рексофт» активно участвует в подготовке ИТ\-специалистов для сокращения существующего дефицита кадров. Для этого компаний сотрудничает с образовательными учреждениями: университетами в городах своего присутствия, EdTech онлайн\-формами, профессиональными конкурсами и программами, нацеленными на IT\-тематику.


В этот раз ведущий эксперт «Рексофт», руководитель практики Java Зураб Белый, стал одним из преподавателей на комплексном годовом курсе Skypro по подготовке Java\-разработчиков с дипломом государственного образца. В рамках сотрудничества с онлайн\-университетом эксперт создал и будет вести несколько модулей программы, а именно: работа с кодом, посвящённый web\-разработке, часть блока по продвинутой работе с кодом, подготовка к техническому собеседованию, командная работа (организация работы в команде, softskills, инструменты эффективного взаимодействия), механика работы над реальными задачами в проекте.



> «Передача опыта людям, которые решили связать свою жизнь с IT, часть моего жизненного кредо. Во всех своих программах я обязательно даю большую долю реального практического опыта, который востребован именно сейчас в индустрии», — прокомментировал свою работу для Skypro **Зураб Белый, руководитель практики Java компании «Рексофт»**.
> 
> 
> «Профессия Java\-разработчик занимает 1 место в топ\-50 востребованных профессий дистанционной работы в IT по данным исследования Министерства экономического развития РФ. Это одна из самых сложных позиций для рекрутера: ​​на рынке дефицит специалистов. Поэтому на толковых начинающих разработчиков всегда есть спрос. Формируя нашу программу, мы попытались соблюсти баланс между теоретическими и практическими знаниями. Это нам помогают делать действующие эксперты рынка, такие как Зураб Белый», — поделилась информацией **Екатерина Гаврилова, продюсер курса онлайн\-университета Skypro**.


***Справка**:*


*Зураб Белый, руководитель практики Java компании «Рексофт». Закончил Воронежский Государственный Университет по специальности «Математическое и компьютерное моделирование». В IT с 2009 года. Свою первую программку написал в 12 лет и с тех пор разработка стала страстью его жизни. Прошел путь от фрилансера до руководителя подразделения. Выступает на конференциях, пишет статьи и преподает Java\-разработку.*
